# Extensiones-de-archivo-y-Markdown

1) ¿Qué significa extensión de archivo?

Son las letras que aparecen después del punto en el nombre de un archivo. Es para indicar qué tipo de archivo es y que programa puede abrirlo o ejecutarlo.

Ejemplos: 

.txt (archivo de texto)

.jpn (imagen)

.html (página web)

.java (archivo de código java)

.pdf (documentos)

.py (código escrito en Python)



2) ¿Qué es Markdown?
   

Es un lenguaje o forma de escribir texto con formato utilizando símbolos.



3) ¿Para qué se usan archivos?
   

Se utilizan para poder almacenar información digital en un computador.
